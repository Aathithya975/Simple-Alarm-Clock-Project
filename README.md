A simple alarm clock application built with Python and Tkinter — set, manage, and trigger alarms with a clean GUI.

Simple Alarm Clock is a Python mini project featuring a GUI-based alarm clock 
built using Tkinter. Users can set a custom alarm time and get an audio alert 
when the time is reached.

Features:
- ⏰ Set alarm by entering HH:MM:SS
- 🔔 Audio alert when alarm triggers (using pygame / playsound / winsound)
- 🕐 Live digital clock display
- 🔁 Simple and beginner-friendly code structure

Built with: Python, Tkinter, datetime, time

Developed a Simple Alarm Clock desktop application as a Python mini project.
The app allows users to set a custom alarm time through a GUI, displays a 
live clock, and triggers an audio alert at the specified time.

Tech Stack: Python, Tkinter, datetime, pygame / playsound
